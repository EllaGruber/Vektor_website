# Vektor Project — Research Website

Website for the Vektor research project, led by **Ella Gruber** (PhD student, Karolinska Institutet) under the supervision of **Prof. Torkel Klingberg**, co-supervised by **Prof. Ola Helenius**, and in collaboration with **Prof. Bruce McCandliss**.

Vektor is an adaptive mathematics training app developed at Karolinska Institutet, combining visuospatial working memory training with number-line tasks to improve mathematical ability in children aged 6–9. This project deploys the original Vektor app to Grade 1 teachers across Swedish schools as part of a research intervention.

## What's in this repo

A single self-contained `index.html` file — no frameworks, no dependencies, no build step needed. Open it in any browser and it works.

## Features

- Bilingual (English / Swedish) — toggle in the top bar
- Teacher portal — school sign-up and student ID generation
- Math anxiety questionnaire (online, printable, and spoken modes)
- Research section with linked peer-reviewed papers
- Researcher team profiles
- Announcements board
- GDPR & personal data information
- Contact form
